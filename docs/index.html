The parts

Holder

Snap-fit device holder. Carries the keyed diamond boss that plugs into any Strong Vibes Connect socket. Best in flexible TPU.

Female

Socket 180

The default keyed socket. One diamond pocket — the boss seats in a single orientation (mod 180°) for a rigid, no-wobble mount.

Female

Socket 30

Star socket — the diamond repeated every 30°, so the boss can seat at twelve angles for stepwise rotational mounting.

Male

Test boss

A stand-in for the holder boss — the diamond on a round base with a real 1/4"-20 thread. Print it with a socket to bench-test the fit.

Strong Vibes Connect v1.0.0

One mating interface, defined once, imported by every part — so the connection can never drift.

How it mates

A keyed diamond boss seats flat into a matching diamond socket and is clamped by a 1/4"-20 camera/tripod screw down the joint axis. The diamond is keyed (long axis ≠ short axis), so it seats in one orientation — rigid and repeatable.

The frozen standard

Long half-diagonal DIA_A = 11.322 mm, short DIA_B = 7.759 mm, grip band BAND_DEPTH = 2.0 mm. Read the full Strong Vibes Connect standard →

Print it

No CAD needed for this path.

  • Preview — spin the part above to check it's what you want.
  • Get an export — grab a .stl (straight into a slicer) or .step from a part's release/ folder. Blessed exports are added as they're cut, so if a folder is empty, build one from source (instructions).
  • Slice — open it in Bambu Studio or PrusaSlicer.
  • Material — Bambu TPU for AMS (68D) for the flexible snap-fit holder; PLA Tough as a rigid alternative for sockets.
  • Assemble — one 1/4"-20 screw clamps the joint. With the socket tapped, the screw stays captive.
